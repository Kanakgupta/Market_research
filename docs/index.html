Technology & Standards Radar

Current vs. next spec version and in-flight features per domain — direct input for roadmap planning.

Bluetooth

Now: Bluetooth Core Specification 6.3 (adopted) Next: Not yet announced (6.3 is the latest adopted release)
  • Channel Sounding Inline PCT Transfer (BT 6.3) — analog phase pre-compensation for faster, lower-overhead ranging
  • Channel Sounding PHY-specific RTT Accuracy (BT 6.3) — per-PHY ranging precision (LE 1M/2M)
  • Running Out of Bits (BT 6.3) — expanded HCI command/event bitmask for future feature growth

AIROC CYW20829 ships Channel Sounding (PBR/RTT baseline from BT 6.0); CYW55513 covers LE Audio. Track BT 6.3 Channel Sounding refinements (Inline PCT Transfer, per-PHY RTT) for next silicon revs.

Wi-Fi

Now: Wi-Fi 7 (802.11be) certified Q1 2024 Next: Wi-Fi 8 (802.11bn) draft, target 2028
  • Wi-Fi 7 MLO mass deployment in routers/clients
  • Wi-Fi 8 UHR — Ultra-High Reliability (deterministic, mobility)
  • AFC (Automated Frequency Coordination) for 6 GHz standard power

AIROC CYW5557x (Wi-Fi 6E + BT 5.4) competing with Qualcomm/MediaTek/Realtek. No Wi-Fi 7 announced yet — gap vs Synaptics Veros/MediaTek Filogic 380.
